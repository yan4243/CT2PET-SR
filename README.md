# CT2PET-SR

Official implementation of **CT-Conditioned Diffusion Prior with Physics-Constrained Sampling for PET Super-Resolution**, accepted at MICCAI 2026.

The method combines a CT-conditioned diffusion prior with PSF-aware data consistency and Progressive Physics-Constrained Refinement (PPCR) for PET super-resolution.

## Code release

Training and inference code, configurations, and usage instructions are being organized and will be released in this repository.
